## Context

This is a multi-role retail operations platform (Smartstore turbos-x) built with:
- **Frontend**: Next.js 14 + React 18 + TypeScript + Tailwind CSS

- **Monorepo**: Turborepo + npm workspaces

The system serves 4 business roles: Owner (OWN), Operations Manager (OPS), Inventory Manager (INV), Cashier Supervisor (POS).
